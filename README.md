# Contabilidad App

Aplicación de contabilidad desarrollada en Python, siguiendo el Plan General Contable (PGC 2007) de España.

## Estructura del Proyecto

- `app/`: Código fuente de la aplicación.
  - `models/`: Definición de modelos de datos.
  - `services/`: Lógica de negocio.
  - `repositories/`: Acceso a datos.
  - `ui/`: Interfaz de usuario (pendiente de definición: Tkinter o FastAPI+React).
  - `utils/`: Utilidades generales.
- `database/`: Migraciones y configuración de base de datos.
- `tests/`: Tests unitarios y de integración.

## Configuración

1. Crear un entorno virtual: `python -m venv .venv`
2. Activar el entorno virtual.
3. Instalar dependencias: `pip install -r requirements.txt`
